# file 사용 명령어

> file 사용 명령어 정리



#### 파일 내용 연속 출력하기

```shell
$ cat [옵션] 파일
```

##### 옵션

- `-n`: 행 번호를 붙여서 출력



#### 화면 단위로 파일 내용 출력하기

```shell
$ more [옵션] 파일
```

##### 옵션

- `+행 번호`: 출력을 시작할 행 번호 지정

##### 키

- 스페이스바: 다음 화면 출력
- 엔터키: 한 줄씩 스크롤

- /문자열: 해당 문자열 검색
- q: 종료



#### 개선된 화면 단위로 파일 내용 출력하기

- 스크롤 되어 지나간 내용도 확인

```shell
$ less 파일
```

##### 키

- j: 한 줄씩 다음 행으로 스크롤
- k: 한 줄씩 이전 행으로 스크롤
- 스페이스바, Ctrl + f: 다음 화면으로 이동
- Ctrl + b: 이전 화면으로 이동
