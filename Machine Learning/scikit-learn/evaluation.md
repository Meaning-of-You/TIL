# Evaluation(평가)

> 분류에 사용되는 성능 평가 지표 정리



### 정확도(Accuracy)

실제 데이터에서 예측 데이터가 얼마나 같은지를 판단하는 지표

- 정확도(accuracy)  = `예측 결과가 동일한 데이터 건수 / 전체 예측 데이터 건수`

- 불균형한 레이블 값 분포에서 ML 모델의 성능을 판단할 경우, 적합한 평가 지표가 아님
